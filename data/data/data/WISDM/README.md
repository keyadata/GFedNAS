WISDM dataset for human activity recognition.

The dataset can be downloaded from the official source:
G. Weiss, “WISDM smartphone and smartwatch activity and biometrics dataset,” UCI Machine Learning Repository, 2019. [Online]. Available: https://doi.org/10.24432/C5HK59


Preprocessing:
- Sensor signals segmented using sliding windows
- Window size: 200 samples
- Overlap: 50%
- Features normalised per client
